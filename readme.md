# 会议排座工具

> 单文件 HTML排座工具，双击即可在浏览器中打开。无需安装、无需联网、数据不出本机，专为党政机关、企事业单位会务工作设计。

---

## 简介

会议排座工具是一套适配各类会议的可视化排座工具，替代传统人工排座流程，
实现**座位图模板识别、名单管理、排列规则选取、自动排布、模板填充、终稿导出**
的一体化管理，大幅提升会议筹备效率、座位安排准确性与成图美观性。

工具针对政务工作电脑特殊系统及保密需求设计为**单文件 HTML**，
任何带浏览器的电脑都能打开使用，适配 Windows、Mac、国产政务专用系统。

---

## 使用方法

1. 下载 `座位图工具.html`，保存到桌面或任意位置
2. 双击文件，会自动用浏览器打开
3. 直接使用，无需安装、无需联网

**推荐浏览器**：Chrome、Edge、火狐（最新版）

---

## 数据安全

- 所有名单、座位信息**只保存在你自己的电脑里**（浏览器本地缓存）
- 不会上传、不会联网传输、不经过任何服务器
- 关闭浏览器再次打开，数据仍然在
- 换电脑使用时，请用「导出项目」功能保存为文件，到新电脑「导入项目」即可

## 功能

### 一、多分区座位管理

支持将会场划分为多个独立区域，每个区域可独立配置，例如：

- 主席台
- 列席人员区
- 代表团区域
- 嘉宾区域

每个分区均可：

- 独立导入名单
- 独立配置排序规则
- 独立调整座位安排

实现复杂会场的精细化管理。

### 二、多种排座规则

以名单顺序 `a、b、c、d、e、f` 为例：

| 规则 | 单区结果 | 双区结果（中间有过道） |
|------|---------|----------------------|
| 顺序排列 | `a b c d e f` | `a b c 过道 d e f` |
| 逆序排列 | `f e d c b a` | `f e d 过道 a b c` |
| 回字形排列 | `e c a b d f` | `f d b 过道 a c e` |

**说明**

- **顺序排列**：按名单顺序，从左到右、从前到后入座。
- **逆序排列**：单区从右到左倒着排；双区以中间过道为首位向两边排（靠过道的座位是 a，依次向外为 b、c）。
- **回字形排列**：让重要的人坐在中间，常用于**主席台、领导席、贵宾席**等正式场景。
  - 单区：以正中为首位（a 在中央，b 在 a 的右手边，c 在左手边，d、e 在更外侧）。
  - 双区：以中间过道为首位向两边交替展开。

### 三、模板导入排座

适合已有 Word 或 Excel 模板的场景。

**支持**

- Word 模板（`.docx`）
- Excel 模板（`.xlsx`）

**限制**

- 建议模板文件 < 20MB
- 复杂模板（带文本框、SmartArt、艺术字等）可能识别不准确
- 模板内容主要应为基本表格与文字

### 四、自由画布模式

无现成模板时，可从零开始构建会议座位图，支持拖拽创建：

- 主标题、副标题、说明文字
- 主席台
- 单区座位、双区座位

### 五、可视化座位编辑

无论模板模式还是自由画布，均支持：

- 鼠标拖拽换位置
- 右键删除人员、设置特殊占位
- 框选选区划入不同分区
- 调整各组件大小、位置、字体字号
- 实时预览座位图效果

### 六、多格式导出

| 格式 | 操作方式 |
|------|---------|
| Word（`.doc`） | 点击「导出 Word」按钮 |
| Excel（`.xlsx`） | 点击「导出 Excel」按钮（仅 Excel 模板可用） |
| PDF | 点击「打印 / PDF」按钮，在系统打印对话框中选择「另存为 PDF」 |
| PNG 图片 | 自由画布模式下点击「导出 PNG」 |

## 常见问题

**Q：能不能不联网用？**
A：能。第一次打开时会从网络拉取一次字体（不影响功能），之后完全离线可用。如果一开始就是离线环境，字体会自动回退到系统自带的宋体、黑体等，使用不受影响。

**Q：换电脑后名单还在吗？**
A：不在。名单保存在当前浏览器的本地缓存里。换电脑前请用「导出项目」保存为文件，到新电脑「导入项目」即可恢复。

**Q：清空浏览器缓存会丢数据吗？**
A：会。建议重要项目随时「导出项目」备份。

**Q：能多人协同编辑吗？**
A：暂不支持。可以由一人编辑完后通过「导出项目」分享文件给他人。
